监控

可以使用 Prometheus 和 Grafana 监控 IOMesh 集群。

与 Prometheus 集成

如果 Prometheus 由 Prometheus Operator 安装在与 IOMesh 相同的 Kubernetes 集群中，只需修改 iomesh-values.yaml 为：

serviceMonitor:
  create: true

然后升级现有的 IOMesh 集群：

helm upgrade --namespace iomesh-system iomesh iomesh/iomesh --values iomesh-values.yaml

一个 exporter 会被创建，并由 Prometheus 自动收集指标数据。

也可以通过导入 iomesh-prometheus-kubernetes-sd-example.yaml 手动配置 Prometheus。

与 Grafana 集成

下载 iomesh-dashboard.json 并将其导入任何现有的 Grafana。
